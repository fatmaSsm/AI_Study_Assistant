# ✨ AI Study Assistant

An AI-powered desktop study assistant built with Python, CustomTkinter, and Google Gemini API.

Smart Study Assistant helps users analyze educational texts by generating:

* 📄 Clear summaries
* 🔑 Important keywords
* 🎯 Interactive multiple-choice quizzes

The application supports both **English** and **Turkish** with a modern dark-themed interface.

---

# 🚀 Features

## 📄 AI Text Summarization

Generate clean and readable summaries from long educational texts, articles, lecture notes, or book chapters.

## 🔑 Keyword Extraction

Extract the most important concepts and keywords automatically using structured AI responses.

## 🎯 Quiz Generation

Create 5 multiple-choice questions based on the provided text to test understanding interactively.

## 🌍 Multi-language Support

Supports:

* Turkish 🇹🇷
* English 🇬🇧

Users can instantly switch languages from the interface.

## 🎨 Modern Desktop UI

Built with:

* CustomTkinter
* Dark mode design
* Responsive fullscreen layout
* Animated interactive components

## ⚡ Smooth User Experience

* Threaded AI requests
* Non-blocking interface
* Real-time word counter
* Dynamic tabs and quiz feedback

## 🔒 Secure API Key Handling

API keys are stored locally in a configuration file instead of being hardcoded into the project.

# 📦 Installation

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/smart-study-assistant.git
cd smart-study-assistant
```

## 2. Install Dependencies

```bash
pip install customtkinter google-genai pydantic
```

## 3. Run the Application

```bash
python main.py
```

---

# 🔑 Gemini API Setup

This project uses the Google Gemini API.

1. Get your API key from Google AI Studio
2. Launch the application
3. Click the ⚙ Settings button
4. Paste your API key
5. Save and start using the app

Your API key is stored locally on your machine.

# 🧠 How It Works

The application sends the user’s text to Gemini API and uses:

* AI summarization
* Structured JSON responses
* Schema validation with Pydantic

Quiz and keyword outputs are validated before being displayed in the interface.
